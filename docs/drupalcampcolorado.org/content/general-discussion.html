Question about this site's construction

Posted by bcabebe on May 6th, 2010 at 10:43am

Hi I'm new to Drupal and I'd like to know how some of the features of this site were created. I've only been using Drupal for 3 months to create a site for a non-profit organization I belong to, see http://eaa43.org.

1) Is this a completely custom theme or a customizable theme? If customizable, what's the theme name?

2) I like the "Login or Create an Account to participate." links in the Primary(?) menu. Is that a module or custom code?

3) I also like the repeated menu at the bottom of the page, again is that a module or custom code?

Thanks, Brian

About the site

Posted by scott on May 17th, 2010 at 12:00pm

1) The theme is completely custom.

2) The login links aren't actually in the menu. It's custom markup in page.tpl.php, styled to go with the actual menu.

3) The duplicate menus are also in page.tpl.php, called like this:

<?php print theme('links', $primary_links, array('class' => 'links primary-links')); ?>

Feel free to check out the source from the Subversion repository.
